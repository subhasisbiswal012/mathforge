# MathForge — Sound Files

Drop your MP3 or WAV files into this folder with these exact filenames.
The app will play them automatically based on your exam score.

| File             | When it plays         | Suggested sound                        |
|------------------|-----------------------|----------------------------------------|
| `perfect.mp3`    | Score = 100%          | Anime wow / crowd cheer / tada fanfare |
| `excellent.mp3`  | Score 95–99%          | MLG airhorn / victory music            |
| `good.mp3`       | Score 90–94%          | Nice applause / level up sound         |
| `okay.mp3`       | Score 85–89%          | Mild applause / "not bad" sound        |
| `meh.mp3`        | Score 80–84%          | Sad trombone lite / "hmm" sound        |
| `bad_1.mp3`      | Score below 80%       | Bruh sound effect                      |
| `bad_2.mp3`      | Score below 80%       | GTA wasted sound                       |
| `bad_3.mp3`      | Score below 80%       | Windows XP error                       |
| `bad_4.mp3`      | Score below 80%       | Vine boom                              |

## Notes
- Format: MP3 preferred (broadest browser support). WAV also works.
- Volume is set to 70% by default. Edit `results.js` to change it.
- If a file is missing, the app plays nothing (no error shown).
- The four `bad_*.mp3` files are picked randomly below 80%.

## Finding Free Sounds
- [freesound.org](https://freesound.org) — search for meme sounds
- YouTube → download audio using any YouTube-to-MP3 tool
- [zapsplat.com](https://www.zapsplat.com) — large free library

## Renaming
Download whatever sounds you like and rename them to match the filenames above.
